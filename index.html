About Me

Arquiteto de Software, Arquiteto de Soluções Cloud e Arquiteto de Soluções Sharepoint, com especialização MBA em Arquitetura de Soluções pela FIAP.

Dez anos de experiência na indústria de TI, tendo atuado em diversos projetos de grandes instituições como Itaú Unibanco, Citibank, Liberty Seguros, Boa Vista Serviços e Yasuda Seguros.

Excelente skill de comunicação, perfil analítico e hands-on, experiente na coordenação de projetos de desenvolvimento de software utilizando metodologias ágeis como Scrum e times auto-gerenciáveis.

Fluência na língua inglesa e experiência em projetos internacionais, como parceria Suécia e Brasil pelo projeto de P. e D. da Volvo Cars.

Possuindo certificações Microsoft de alto nível, tais como:

  • MCSE - Microsoft Certified Solutions Expert Cloud Platform and Infrastructure
  • MCSD - Microsoft Certified Solutions Developer Web Developer
  • MCSD - Microsoft Certified Solutions Developer App Builder Charter
  • MCTS - Microsoft Certified Technology Specialist
  • MS - Microsoft Specialist - Architecting Microsoft Azure Solutions
  • MS - Microsoft Specialist - Programming in C#
  • Professional Scrum Master certificado pela Scrum.org

Especialista em desenvolver soluções para nuvens Microsoft Azure e Amazon AWS, compreendendo analisar, arquitetar, coordenar e desenvolver as soluções.

Larga experiência no desenvolvimento de soluções utilizando serviços Azure como Azure Service Fabric, Azure Container Service, Azure VMs e Azure VM Scale Set, Azure Cloud Services, Web Apps, Networking, Load Balance, VPN Gateways, Network Watcher, Storage, App Service, DocumentDB, Redis Cache, SQL Database, Cognitive Services (LUIS, Speech Recognition and Face Recognition), Power BI, entre outros.

Experiência na implementação de vários design patterns de nuvem, tais como Cache-Aside, Circuit Breaker, CQRS, Event Sourcing, Health Endpoint Monitoring, Priority Queue, Retry, Runtime Reconfiguration, Throttling entre outros.

Disponibilidade para viagens.

Palestrante do evento The Developers Conference – Brasil em 2015/Florianópolis, contribuidor do projeto open source “Quem-Roubou-Minha-Senha”, entre outros.
